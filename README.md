
# To Run an application using docker


```plaintext
docker compose up --build
```

# To Run the Application Locally

- Configure the `.env` file:
  
```plaintext
DATABASE_USERNAME=USERNAME
DATABASE_PASSWORD=PASSWORD
DATABASE_NAME=NAME
DATABASE_HOST=HOST
DATABASE_PORT=PORT
```

## Installation


```bash
npm install
```

## Runing the app

```bash
npm start
```
